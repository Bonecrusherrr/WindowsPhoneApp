SQLite is a software library that implements a self-contained, serverless, zero-configuration, transactional SQL database engine.  This package contains an extension SDK and all other components needed to use SQLite for Windows Phone 8.1 application development with Visual Studio 2013.  Further information, including complete documentation, may be found at the official web site "https://www.sqlite.org/".
